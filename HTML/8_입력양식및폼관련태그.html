폼 관련 태그

form태그 내의 input 요소들에 사용자가 입력한 정보들을
서버로 넘기면서 요청하는 역할을 수행하는 태그이다.

<form action="" method="" >
* action 속성 : 폼 내에 입력된 값들을 전달할 서버에 대한 경로 제시
* method 속성 : 요청전송 방식을 지정하는 속성 - get(기본값 )/ post
> get 방식 : 요청시 사용자가 입력한 값들이 url에 노출되는 방식 (전달값에 크기 제한)
> psot 방식 : 요청시 사용자가 입력한 값들이 url에 노출이 안되는 방식 (전달값에 크기 제한없음)

action에 지정한 경로로 사용자가 입력한 값을 key=value 세트로 전달

체크박스와 라디오버튼 관련 input 태그들

라디오버튼 (type="radio)

- 제시한 여러개의 값들 중에서 한개만 선택해야만 할때 (즉, 하나의 그룹으로 사용할 경우 name속성값이 같아야한다!>

- 선택된 값(checked)만 넘어감

- 사용자가 선택한 값을 넘기고자한다면 선택했을 때의 어떤값을 넘길건지 value속성을 이용해서 명시해둬야된다!
즉, 라디오버튼은 key와 value값을 동시에 제시해야된다!

성별 :

체크박스 (type="checkbox")

- 제시한 여러개의 값들 중에서 여러개 선택가능 할 경우

- 체크박스는 key값이 중복돼도 가능하다(여러개를 체크할수있으니까)
