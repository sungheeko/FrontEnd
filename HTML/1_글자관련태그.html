문단을 나누는 태그 (p, pre)

문단 영역을 나누는 태그로느 p태그와 pre태그가 있다. p 태그는 문단 영역을 나누는 태그지만 줄바꿈을 하고자 한다면
별도의 태그를 기술해야됨. 뿐만 아니라 한개의 공백만 표시하기 때문에 추가적인 공백을 넣고자      한다면 특정구문을 기술해야된다. 

            pre 태그는 여러칸 띄어쓰기       혹은 줄바꿈 등을 포함하여
            입력한 내용 있는 그대로 표현하는 태그이다.

기타 글자 관련 태그

b : 글자를 굵게 표시하는 태그
strong : 글자를 굵게 표시하는 태그
em : 글자를 기울이는 태그
mark : 형관펜 효과를 나타내느 태그
u : 글자에 밑줄긋는 태그
s : 글자에 취소선을 넣는 태그
small : 글자를 작게 표시하는 태그
big : 글자를 크게 표시하는 태그
기본 글자에 아래첨자를 나타내는 태그와 윗첨자를 나타내는 태그
IoT란 각종 사물에 센서와 통신기능을 내장해 인터넷에 연결하는 기술이다.

태그들 중첩 사용

태그들 간에 중첩으로 사용가능하다.
굵은글자를 사용할 수도 있고, 기울이거나,
굵게 표시하면서 취소선도 넣을 수 있음.
